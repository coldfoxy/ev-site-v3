# ЭВ — Экономика впечатлений (версия 3, газетная)

Третий вариант сайта-визитки «ЭВ» в редакционной (editorial / газетной) стилистике: кремовая «бумага», крупный шрифт с засечками (Cormorant) в бронзе, мелкие подписи без засечек (Akt), тонкие линейки, списки-строки, минимум картинок.

Статичный сайт — подходит для GitHub Pages.

## Структура

```
docs/
  index.html        ← страница: все тексты подключаются здесь
  css/style.css     ← оформление (цвета и шрифты — в блоке «ПЕРЕМЕННЫЕ»)
  js/main.js        ← появление блоков при прокрутке + мобильное меню
  fonts/            ← фирменные шрифты (Akt, Aeonik Pro, Cormorant Infant)
  img/              ← логотип (SVG) и одно фото проекта
```

Файлы прокомментированы построчно.

## Публикация на GitHub Pages

1. Создайте на GitHub **пустой** репозиторий (без README/.gitignore/license).
2. Подключите и отправьте:
   ```bash
   cd ~/Desktop/ev-site-v3
   git remote add origin https://github.com/ВАШ_ЛОГИН/ИМЯ-РЕПО.git
   git push -u origin main
   ```
3. Settings → Pages → Branch: **main**, папка **/docs** → Save.
4. Сайт появится на `https://ВАШ_ЛОГИН.github.io/ИМЯ-РЕПО/`

## Частые правки

| Что | Где |
|---|---|
| Тексты | `docs/index.html` |
| Цвета и шрифты | `docs/css/style.css`, блок «ПЕРЕМЕННЫЕ» |
| Акцентный цвет (бронза) | переменная `--accent` |
| Цвет «бумаги» | переменная `--paper` |
